H5新特性

1.语义化标签，例如header，footer，section，article等

语义化的好处

标签名贴近功能，代码分块清晰，便于维护。

页面结构清晰，便于浏览器和搜索引擎解析。

利于搜索引擎爬取；利于SEO(搜索引擎优化)，提高自然排名增加流量。


2.音视频，audio video

audio和video标签能够很容易的输出音频或视频流，提供便利的获取文件信息的API


3.画布 canvas

Canvas API 提供了一个通过JavaScript 和 HTML的canvas元素来绘制图形的方式。它可以用于动画、游戏画面、数据可视化、图片编辑以及实时视频处理等方面。


4.表单增强

calendar、date、time、email、url、search


5.本地存储localStorage,sessionStorage

localStorage:一直保存在浏览器里，除非用户手动清除

sessionStorage:从进入网站到浏览器关闭所经过的这段时间，也就是用户浏览这个网站所花费的时间。session对象可以用来保存在这段时间内所要求保存的任何数据。基于会话，关掉页面或浏览器就会结束生命周期。

大小都是5M


6.web worker，websocket

webworker：开启一个子线程，为 JavaScript 创造多线程环境，允许主线程创建 Worker 线程，将一些任务分配给后者运行。在主线程运行的同时，Worker 线程在后台运行，两者互不干扰。用于耗时操作不影响到主线程UI流畅。

websocket：客户端与服务端双向数据通信协议，用于解决HTTP协议通信只能由客户端发起的缺陷

rem等比缩放

设置根节点html的font-size为10px或100px

后续单位设置成rem，基于根节点的值等比例关系1rem=一倍的font-size

多设备的等比例适配，通过js动态改变根节点的font-size值来控制布局的单位比例关系

docement.documentElement.style.fontSize=document.documentElement.clientWidth/375*100+"px

375是你基于设备宽度的基准值

计算其他设备的宽度是你基准设备宽度的多少倍

把这个倍数算给font-size的值，实现其他设备宽度是基准设备宽度的多少倍来改变局部的比例关系

vw vh单位

100vw = 百分比的适口宽度

可以省去js计算rem的font-size

例如375px的基准值 width 375px=100vw

1px =100/375 约等于0.2667 vw
